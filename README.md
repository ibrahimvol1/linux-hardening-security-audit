# Linux Hardening and Security Audit

This repository documents an end-to-end Linux hardening and security audit performed on Ubuntu 24.04.
It contains configuration changes, scripts, Lynis reports, and step-by-step notes for reproducibility.

## Contents
- `hardening-steps.md` — step-by-step actions performed
- `lynis-before.txt` — Lynis output before hardening
- `lynis-after.txt` — Lynis output after hardening
- `suid-files.txt` — SUID files list
- `scripts/hardening.sh` — small automation script
- `docs/` — optional screenshots and additional notes

## Purpose
Prepared as a portfolio project for SOC / Blue Team internships.
